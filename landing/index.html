From Legacy SQL to Modern Type-Safety

SunSetter converts complex SQL relationships and constraints into beautiful, readable Convex code.

  • Enums auto-detected & mapped to v.union()
  • Primary/Foreign keys mapped to v.id()
  • JSONB fields mapped to v.any() or v.array()
schema.ts
export default defineSchema({
  users: defineTable({
    name: v.string(),
    age: v.number().gte(18),
    role: v.union(
      v.literal("admin"),
      v.literal("user")
    )
  }).index("by_role", ["role"])
});
